# 2. 基于定时器的心跳机制 demo 实现

  • 代码逻辑

    A 定时请求数据 ==> 发送请求给 B 的接口 ==> B 的接口接收信息，返回对应的信息 ==> A 处理返回的信息

    这个过程不需要 A、B 两者用 socket 连接，实现了解耦

  • 定时

    参考博客： https://blog.csdn.net/u010963948/article/details/52946268
